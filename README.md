
# Rect Basic CRUD APP

## Run Locally

Clone the project

```bash
  https://github.com/NAW2005/react-basic-crud.git
```

Go to the project directory

```bash
  cd my-project
```
Getting Started
```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
```
Open http://localhost:5173/ with your browser to see the result.
